
==============================================
🔧 CONDITIONS DE COUPE — APPLICATION PYTHON
==============================================

📦 Nom de l’exécutable :
    → APPP.exe

📂 Dossier de travail :
    → Ce fichier doit être placé dans le dossier contenant :
        - app.py
        - conditions_coupe_sandvik.json
        - app_icon.ico (facultatif)

🚀 Fonctionnalité :
    Cette application permet de calculer automatiquement les conditions de coupe 
    (n, Pc, Mc, etc.) à partir des recommandations SANDVIK et selon le type d’opération.

✅ Prérequis :
    - AUCUN si vous utilisez l’exécutable .exe fourni.
    - Si vous exécutez via Python : Python 3.13 + Streamlit installé

📄 Mode d’emploi :
    1. Double-cliquez sur le fichier `ConditionsCoupe.exe`
    2. Le navigateur va s’ouvrir automatiquement avec l’interface de calcul
    3. Sélectionnez une plaquette, entrez les paramètres et visualisez les résultats

🛠️ Dépannage :
    ❌ Si rien ne s’ouvre :
       - Vérifiez que vous n’avez pas supprimé `app.py` ou `conditions_coupe_sandvik.json`
       - Vérifiez que l’Antivirus n’a pas bloqué le .exe
